# Student Performance Prediction

## Overview
This project predicts student marks based on study hours using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Features
- Reads student data from CSV
- Visualizes study hours vs marks
- Trains a Linear Regression model
- Predicts marks for new study-hour values
